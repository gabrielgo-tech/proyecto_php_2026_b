# Proyecto PHP.

## Descripcion.
Este es un proyecto de tarea para el TDS Galileo, que consiste en realizar una aplicación en el lenguaje de programación PHP que muestre en pantalla un “Hola mundo, soy [nombre del estudiante]” e incluir una imagen a discrecion, y yo escogi una pintura, que pertenece a Jules Breton y se llama Calling In The Gleaners.

## Prerequisitos.
1. Instalar Visual Studio Code
2. Instalar git
3. Instalar apache2
4. Instalar php
5. Instalar el compilador de C (build-essential)

## Pasos para su ejecucion.
Clonar el repositorio:
Abre una terminal y dirígete a la carpeta raíz de tu servidor web (por ejemplo, /var/www/html en Linux o la carpeta de XAMPP/Laragon) y ejecuta:

Bash
git clone https://github.com/gabrielgo-tech/proyecto_php_2026_b.git
Asegurar permisos (Solo si usas Linux/Apache2):
Asegúrate de que el servidor tenga permisos para leer la carpeta del proyecto:

Bash
sudo chown -R www-data:www-data proyecto_php_2026_b
Iniciar el servidor:
Verifica que el servicio de Apache esté corriendo.

Acceder desde el navegador:
Abre tu navegador de preferencia e ingresa la siguiente URL:
http://localhost/proyecto_php_2026_b/
